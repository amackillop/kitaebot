# Kitaebot

An autonomous agent inspired by [nanobot](https://github.com/AbanteAI/nanobot) and a simpler alternative to [OpenClaw](https://github.com/openclaw/openclaw).

Currently a hobby project to understand how these things work.

## Design Goals

- **Security-first**: Strong sandboxing that's simple and easy to verify
- **NixOS-native**: Built for easy deployment and development on NixOS systems
- **Minimal complexity**: As simple as possible, no more

Powered by Nix and Rust.
